## 🏗️ Metodologia de Estudo & Produção

Este projeto utiliza um pipeline de documentação moderna para garantir a melhor retenção de conteúdo:

1.  **Brainstorming:** Notas estruturadas e rascunhos elaborados originalmente no **Microsoft OneNote**.
2.  **Refinamento via IA:** Processamento e estilização dos resumos realizados pelo **Claude (Anthropic)**, garantindo objetividade e formatação técnica superior.
3.  **Desenvolvimento:** Estruturação em HTML5/CSS3 com foco em design minimalista e responsivo.
4.  **Deploy:** Publicação automatizada via **Netlify** para visualização rápida em qualquer dispositivo.
